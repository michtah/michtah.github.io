Production Diary - Week 1

Made on: February 10th 2026

Hello there! This is the first entry in my production diary. I hope you enjoy this little diary of what I made this week, and what I learned.

This week, I made this very website. It was an interesting experience. I added this portfolio as a link to my main page. I learned how to use the header and nav tools more effectively. It really taught me a lot. So far, my proudest achievement is the look of it. We haven't started learning CSS properly just yet, but I remember some things from when I was learning web development on my own in high school. I wanted to create a blog back then, so I took up web development and some CSS on the way. The hardest part was honestly organising the code itself. I really am not used to a language where you have to type in everything and you can't take off large indented chunks into another function. It's quite interesting. HTML organisation is hard. In terms of anything theoretical I learned, I think I'd say how webpages are organised and how their structure and presentation data is entirely separate. It's quite interesting. The same webpage with the same structure can be presented a billion different ways. I learned how to set an ID for each item in the webpage. Next week, I want to work on my other sub-site, and add an article to the large numbers blog.

On the other end, I also am trying to learn some assembly. Currently I'm learning NASM, as it appears to be the one we're focusing on in our classes. I've found a pretty good website to simulate running NASM. I don't know if I want to become proficient in assembly yet, however. We'll see.
