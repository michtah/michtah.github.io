Weekly Diary

Week 2 - Redesigning my Portfolio

This week, I rewrote my portfolio page from scratch. This was an interesting challenge as I have learned a lot of CSS between my previous attempt and this current attempt. I learned how to style pages, how to use the rem scaling, how to set background colors, and how to set the different states of links and buttons. I also learned how to make these cool floating sections, though this is not in the curriculum, I just thought they looked cool.

The big challenge was just getting things to look right. You have to set a lot of styling, even for such a small page, to get everything looking exactly how you want. It's quite difficult. Another thing was deciding how much detail to add in my profile. I don't want to add too much, as this is a public website, but I think this was a good middle ground.

I'd like to eventually come back and restyle/rewrite some parts of the page, or maybe integrate a different font, or include some images. For now though, I'm quite happy with it.
